# Классификация отходов по изображениям

## Бизнес-постановка задачи

В мире, где переработка отходов играет важную роль в защите окружающей среды, сортировка мусора является ключевым элементом в повышении эффективности перерабатывающих заводов.
Допустим, что один из подобных заводов мог заказать подобное IT-решение ввиду стремления руководства к улучшению экологической обстановки.
Задача проекта — автоматически определять тип отходов по изображению,
чтобы упростить и автоматизировать процесс сортировки.

---

## Постановка задачи в терминах DS

- **Область:** Computer Vision  
- **Тип задачи:** Многоклассовая классификация изображений  
- **Вход:** изображение объекта  
- **Выход:** класс отхода (plastic, glass, metal, paper, cardboard, trash)  
- **Метрика:** accuracy  

---

## Данные

Используется готовый, предварительно обработанный датасет изображений отходов, найденный на Kaggle.
Дополнительная предобработка не выполнялась.
https://www.kaggle.com/datasets/asdasdasasdas/garbage-classification/data

Датасет был разделён на:
- обучающую выборку
- валидационную выборку
- тестовую выборку

---

## Модели

### Baseline
В качестве бейслайн-модели используется простая сверточная нейросеть (Simple CNN),
обучаемая с нуля. Она служит точкой отсчёта для оценки качества.

### Предобученная модель
В качестве основной модели используется **ResNet-50**, предобученная на ImageNet,
с заменой последнего слоя под количество классов в задаче.
https://huggingface.co/microsoft/resnet-50

ResNet-50 была выбрана, потому что это надёжная и широко используемая модель для классификации изображений.
Она уже обучена на большом наборе данных (ImageNet), поэтому хорошо распознаёт визуальные признаки и даёт более высокую точность, чем простая модель, обучаемая с нуля.

---

## Что было сделано

- Загружен готовый датасет изображений
- Выполнено разделение данных на train / val / test
  (в равных пропорциях для каждого класса в датасете)
- Реализована бейслайн-модель (Simple CNN)
- Подключена и обучена предобученная ResNet-50
- Проведено сравнение качества моделей
- Визуализированы метрики обучения
- Продемонстрированы предсказания модели на тестовых изображениях
  (и случайных стоковых)

---

## Результаты

Предобученная модель ResNet-50 показывает более высокую точность по сравнению
с бейслайн-моделью, что подтверждает эффективность transfer learning
для задач классификации изображений.
